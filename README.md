## Ejecutar localmente
Basta con abrir `index.html` en el navegador.

## Deploy recomendado
### GitHub Pages
Como este proyecto es estático, la opción más simple y correcta es **GitHub Pages**.

Pasos dentro del repo en GitHub:
1. Ir a **Settings**.
2. Entrar a **Pages**.
3. En **Build and deployment**, elegir **Deploy from a branch**.
4. Seleccionar la rama **main** y carpeta **/** (root).
5. Guardar.

Luego GitHub te dará una URL pública.
